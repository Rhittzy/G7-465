Welcome to G.E.N



This website is an experimental version of our project



The main goal of our project is to develop a user friendly website that engages users along with allowing them to browse, create, and review events efficiently. Ironically, myinvolvement is currently limited to allowing only organizations to post events which leaves other non org related events undermined. As a result people resort to social media as a way of spreading awareness for better or worse. We think by allowing all students to create events permits for a more equal and involving environment.



Another aspect we would like to allow users to do is leave reviews or even offer surveys based on how an event went. This would let users post critical feedback expressing their experience. The final objective related to the surveys is being able to track and display analyzed data of the most popular event days, locations, and engagement levels.



If time allows we would also like to implement a room booking service which would compliment event booking in the sense that people planning events need a space to host the event. Ideally this would eliminate the need for two separate websites by combining the two.
